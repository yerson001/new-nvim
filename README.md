# new-nvim

## Install 

Download https://github.com/neovim/neovim/releases

.appimage for linux
.deb for debian ... etc

## Move 
sudo mv ~/Descargas/nvim.appimage /opt/nvim.appimage

## Permision
sudo chmod +x /opt/nvim.appimage

## Target Link
sudo ln -s /opt/nvim.appimage /usr/local/bin/nvim

## Check
nvim --version
